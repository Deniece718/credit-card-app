# credit-card-app

## Introduction

This is a modern **Credit Card Dashboard** app mainly using **React Native**, **Expo**. This app allows users to check their companies' credit card status, view invoices, transactions, card detail and manage their spending limit.

## Tech Stack

* **React Native** – For building native apps
* **Expo** – For simplifying the React Native development
* **React Hooks** – For managing component state and side-effects
* **TypeScript** – Type safety and tooling
* **GitHub** – Deployment

## Features

* **Company/Card Select**
  Display the selected company and credit card's invoices, transactions, card status, and spending condition.
  
* **Activate/Deactivate Card**
  Allow user to activate and deactivate card.

* **Update Card Limit**
  Allow user to change the card's spending limit.

* **Update Card Limit**
  Allow user to submit support.

* **Real-Time Data**
  Sync your company's credit card in real-time with Express Backend.

* **Responsive Design**
  Use Expo's built-in features to create a mobile-first, responsive design.


### Clone and Run

```bash
git clone https://github.com/Deniece718/credit-card-app.git
cd credit-card-app
npm install
npm start
```
The app will be available at: [http://localhost:8081](http://localhost:8081) (for Expo development)
